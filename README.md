# ğŸ’» Aplikasi Pendaftaran Siswa (CRUD PHP + MySQL)

Ini adalah proyek aplikasi web sederhana yang dibuat dengan PHP natif untuk mengelola data pendaftaran siswa baru. Aplikasi ini mengimplementasikan fungsionalitas **CRUD** (Create, Read, Update, Delete) secara penuh.

Proyek ini menggunakan PHP natif tanpa *framework* dan database MySQL, cocok untuk latihan dasar pengembangan *backend* web.

---

## ğŸš€ Fitur Utama

* **Create:** Menambahkan data siswa baru melalui formulir (`form-daftar.php`).
* **Read:** Menampilkan semua data siswa yang telah terdaftar dalam bentuk tabel (`list-siswa.php`).
* **Update:** Mengedit dan memperbarui data siswa yang sudah ada (`form-edit.php` & `proses-edit.php`).
* **Delete:** Menghapus data siswa dari database (`hapus.php`).
